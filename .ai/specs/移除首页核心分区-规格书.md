# 移除首页核心分区 技术规格书

## 目标
- 删除重复且占据整屏的“核心分区”，让首屏后直接进入近期爆款。

## 影响范围
| 文件 | 原因 |
|---|---|
| `index.html` | 删除核心分区结构和分隔线，修正导航锚点 |
| `css/style.css` | 删除核心分区及其响应式无用样式 |

## 实施步骤
1. 删除 `featured-section#categories` 整块 HTML 和紧随其后的分隔线。
2. 导航栏原“分区”入口改为“爆款”，锚点指向 `#games`。
3. 删除 `.featured-*` 与 `.section-divider` 样式，包括响应式覆盖。

## 验证方式
- 全局检查不存在 `#categories`、`.featured-*` 和 `.section-divider` 残留。
- 验证 `#games`、`#free-section` 导航锚点仍存在。
- 本地文件模式渲染检查首屏后直接进入近期爆款。

## 回滚方案
- 恢复删除的 HTML 分区、导航锚点和对应 CSS。

## 目标编辑文件清单
- `index.html`
- `css/style.css`
